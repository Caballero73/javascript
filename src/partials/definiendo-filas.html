Definiendo filas

Una vez definidos el tipo de display y las columnas, debemos completar la estructura básica de la grilla con las filas (eje vertical).


• grid-template-rows establece el tamaño de las filas (eje vertical).

Combinado las columnas y las filas en el siguiente ejemplo tenemos una cuadricula con 3 columnas (la primera con 20% de ancho, la segunda con 50% de ancho y la tercera con 30% de ancho) y con 3 filas (la primera con 15rem de alto, la segunda con 5rem de alto y la tercera con 10rem). Ahora, de acuerdo al número de ítems (elementos hijos) que tiene el contenedor grid, tenemos una cuadrícula de 3x3 elementos. Como el número de ítems es par, la última celda de la cuadrícula se queda vacía.

      HTML
      <div class="padre">
        <div class="hijo">Contenido... #1</div>
        <div class="hijo">Contenido... #2</div>
        <div class="hijo">Contenido... #3</div>
        <div class="hijo">Contenido... #4</div>
        <div class="hijo">Contenido... #5</div>
        <div class="hijo">Contenido... #6</div>
        <div class="hijo">Contenido... #7</div>
        <div class="hijo">Contenido... #8</div>
      </div>

      CSS
      .padre {
        display: grid;
        grid-template-columns: 20% 50% 30%;
        grid-template-rows: 15rem 5rem 10rem; ◀ Esto define las filas
      }

      .hijo {
        background-color: rgb(255, 243, 204);
        border: 1px solid #777;
        padding: 10px;
        text-align: center;
      }
