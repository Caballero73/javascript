Construir objetos en JavaScript

Para JavaScript, como lenguaje de programación, casi TODO es un objeto. Números, cadenas, arrays, funciones, datos booleanos, fechas y expresiones regulares son percibidas como objetos o pueden ser tratados como tal.

Para entender mejor el concepto de objeto imaginemos que acabamos de comprar un móvil nuevo (el móvil es el objeto). Tiene propiedades como color, peso, tamaño, tamaño de pantalla y marca; también tiene métodos como llamar, tomar una foto, encendido y apagado. Ahora, expliquémoslo a nivel de programación:

  • Los objetos son una instancia de una clase
  • Las propiedades son una característica, dada en un valor; ya sea tipo String (cadena), matemática, etc
  • Los métodos son una capacidad; y para realizarlos utilizamos una función (function)

En JavaScript encontramos 3 maneras de crear objetos:

Crear un objeto literal Crear un objeto usando la palabra new Crear un objeto haciendo una función constructora

Crear un objeto literal

Un objeto literal es la manera más simple de crear un objeto. Se enmarca dentro de corchetes y se describe en pares de (clave: valor).

      
      var movil = {
        Marca : "Alcatel", 
        Peso : "5 onzas", 
        pantalla : "5 pulgadas", 
        Color : "Negro"
      };

Crear un objeto usando la palabra new

El operador new crea una instancia de un tipo de objeto a partir de una función constructora. Recordemos que en JavaScript un constructor es una función.


      var movil = newObject();
      movil.Marca = "Alcatel";
      movil.Peso = "5 onzas";
      movil.Pantalla = "5 pulgadas";
      movil.Color = "Negro";

Crear un objeto haciendo una función constructora


      function movil(marca, peso, pantalla, color) { 
        this.Marca = marca;
        this.Peso = peso;
        this.Pantalla = pantalla;
        this.Color = color;
      }
      var compra = new movil("Alcatel", "5 onzas", "5 pulgadas", "Negro");

Una función constructora crea una copia de un objeto específico. En otros lenguajes de programación son conocidas como clases. En JavaScript encontramos dos maneras diferentes de acceder a las propiedades de nuestro objeto:

  • Marca;
  • movil[“Marca”];

Su sintaxis es distinta, pero realizan la misma acción.

Ejemplo:


      var movil = {  
        Marca : "Alcatel", 
        Peso : "5 onzas", 
        pantalla : "5 pulgadas", 
        Color : "Negro"
      };
      document.getElementById("elementoprueba").innerHTML = movil.Marca + " --- " + movil["Marca"];

Ahora vamos a agregar métodos. Existen diversas maneras de realizar este proceso, pero tomaremos de ejemplo la manera más simple: utilizando la palabra this. Se puede utilizar en un método para referirnos al objeto actual.

        
      var movil = {
        Marca : "Alcatel",
        encendido : function(c) {
          return "Mi movil marca" + " " + this.Marca + " " +  "esta encendido"; 
        }};
        document.getElementById("elementoprueba2").innerHTML = movil.encendido();
