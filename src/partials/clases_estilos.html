Manipular Estilos y Clases

La forma en que se ven nuestros documentos HTML o XML, o mejor dicho como el navegador representa los elementos que los conforman, se define mediante CSS o Cascading Style Sheets (lógicamente esto ya lo sabemos desde hace tiempo).

Estos estilos son instrucciones que pueden estar directamente en el código HTML o en archivos externo a los que se hace referencia (Archivos CSS). Ahora bien, una gran limitación del CSS es su rigidez, ya que una vez cargada la web no se pueden cambiar. Afortunadamente Javascript por ser ser un lenguaje interactivo nos permite realizar cualquier tipo de modificación en el estilo original de un elemento (nodo) y esto sin que sea necesario volver a cargar la página desde el servidor.

Nos es posible modificar el tamaño, estilo, tipografía y todos los demás atributos del texto, además el color del fondo, estructura de tablas, listas, en fin el estilo de todos los elementos que componen la página.

Para que el usuario realice dichas modificaciones puede utilizarse cualquier evento, desde un clic del ratón, hacer scroll o cualquier otro método con que se pueda trasmitir una instrucción, inclusive puedeser de forma automática al cargar la web.

Cambiar un atributo de estilo con JavaScript

En Javascript se utiliza para cambiar el estilo de los elementos o nodos el objeto style (Estilo), este es una declaración independiente y puede ser accedido de varias formas, desde el document o desde el elemento individual al que va dirigido.

La sintaxis es la siguiente:

      document.body.style.fontSize="24px"
      // Este método permite aplicar el estilo 
      de forma general a todo el documento.   


      document.getElementById("idElemento").style.fontSize="24px"
      // Este método es más específico, al ser dirigido 
      a un bloque o elemento definido por un identificador ID.

Cambiar una clase con JavaScript

Si lo que deseamos es trabajar con una clase completa y no sólo con atributos de forma particular, la buena noticia es que hoy día todos los navegadores modernos ofrecen una manera muy conveniente de manipular las clases de un elemento a través de JavaScript, esto lo logramos a través de la propiedad classList.

Esta propiedad ofrece varios métodos que nos permiten manipular las clases de un elemento de manera individual:

  • add: añade al elemento un nombre de clase o varios nombres de clases (se pasan separados por comas, como argumentos del método).
  • remove: le quita el nombre o nombres de clase que indiquemos.
  • toggle: este es muy útil porque le quita o le pone el nombre de clase indicado dependiendo de si ya lo tiene o no.
  • contains: devuelve true si el elemento tiene aplicado el nombre de clase que le indiquemos, siendo false en caso contrario.
