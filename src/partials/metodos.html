Métodos

Las cosas que pueden hacer los objetos se llaman métodos. Las personas hablan, las computadoras se rompen y las bicicletas ruedan.

Los objetos de JavaScript también tienen métodos: click() para los botones, open() para las ventanas y el texto puede seleccionarse gracias al método selected(). Los paréntesis indican que estamos refiriéndonos a un método, y no a una propiedad.

Consejo, puede ser útil pensar en los objetos y en las propiedades como nombres, y en los métodos como verbos. Los primeros son cosas, y los segundos, acciones que esas cosas pueden hacer, o que se les pueden hacer a ellas.

Puede poner juntos los objetos, las propiedades y los métodos para obtener una descripción mejor de un objeto, o para describir un proceso.

En JavaScript, todas estas piezas se separan mediante puntos. Esto se denomina sintaxis de punto.

He aquí algunos ejemplos de objetos y sus propiedades escritos de este modo:

  • bicicleta.rueda
  • lobo.garras.delantera.izquierda
  • computadora.disco.cdrom
  • documento.imagen.nombre
  • ventana.estado

Y aquí tiene algunos ejemplos de objetos y métodos escritos con sintaxis de punto:

  • lobo.aullido()
  • document.write()
  • forms.elements.radio.click()

Ejemplo

Un resumen bastante sencillo de este concepto sería que un método es una función dentro de un objeto.


      var persona = {
            nombre: 'Enrique',
            apellido: 'Valerio',
            sexo: 'Masculino',
            edad: 45,
            habilidades: ['Diseño','Dibujo','Desarrollo Web','Exploración',
                  'Vida al aire libre','Aprender haciendo'],
            saludar: function() {
                        console.log('Hola Mundo');
                  }
      };

Para invocar este método se usa persona.saludar() ya que se trata de una función.


this como un método de un objeto

Cuando una función es llamada como un método de un objeto, el this cambia por el metodo del objeto llamado.

En el siguiente ejemplo, cuando objeto.funcion() es invocado, dentro de la función this es ligado al objeto objeto.


      var objeto = {
        edad: 45,
        funcion: function() {
          return this.edad;
        }
      };

      console.log(objeto.funcion()); // Muestra 45

En pocas palabras this permite usar una propiedad de un objeto, declarada dentro de este objeto (ambito local), por un método (función) dentro de este mismo objeto.
